# Reading Curriculum — Top 300 Book Metalist

A **metalist**: a single ranked book list built by merging 14 source lists. Each
source list carries a point value, and every book scores the sum of the points of
all the lists it appears on — so books endorsed by many sources, or by a
highly-weighted source, rise to the top.

## View it online

The interactive list is published with GitHub Pages — see the **Pages** link in the
repository's About section, or `https://<username>.github.io/reading-curriculum/`.

The page lets you:

- search and sort the 300 books,
- filter by difficulty (1–5) or by source list,
- see each book's recommendation sources and a difficulty rating, and
- tick off who in the family has read each book (saved in your browser).

## Files

- `index.html` — the interactive metalist (served by GitHub Pages)
- `Book Metalist.md` — the markdown source of truth, with the full ranked table

## Difficulty

Each book's 1–5 rating is derived from word count, age of the book, and content
type (children's and popular fiction lowest; poetry, epic, philosophy, theology and
science highest).
