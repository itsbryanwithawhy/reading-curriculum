# Reading Curriculum — Top 300 Book Metalist

A **metalist**: a single ranked list built by merging 14 source lists. Each source list is worth a number of points. Every book scores the sum of the points of all lists it appears on, so a book endorsed by many sources — or by one highly-weighted source — rises to the top. Books are sorted highest value to lowest. This file is the source of truth; the companion `Book Metalist.html` is an interactive view of the same data.

## Difficulty rating

Each book carries a 1–5 difficulty rating derived from three factors, consistent with the project brief:

- **Word count** — longer works add difficulty (a ~600k-word novel is a heavier lift than a 30k-word one).
- **Age of the book** — older works add difficulty through archaic language, unfamiliar idiom and assumed context (pre-1700 weighs most).
- **Content** — the kind of book: children's and popular fiction sit low; literary fiction, drama and history sit mid; poetry, epic, philosophy, theology and science sit high. This factor also stands in for external readability context (reading-level guides, scholarly apparatus required, etc.).

Ratings: **1 Easy · 2 Moderate · 3 Challenging · 4 Demanding · 5 Formidable**.
